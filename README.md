# 🚀 智能物联网桌面助手
---
## 项目简介
本项目旨在打造一个“有温度、会思考”的桌面伴侣。通过与嵌入式设备相结合,提供智能化的语音对话与健康管理功能。
---
## 核心功能
* **AI语音管家**：基于语音识别与大模型，支持多轮对话、情感陪伴及硬件指令调度。
* **环境感知**：实时采集温湿度、久坐数据，支持历史趋势分析。
* **智能控制**：风扇 PWM 调速，支持用户进行简单的自定义。
* **健康助手**：久坐监测与提醒、番茄钟专注模式，守护桌面工作者的健康。
* **数据看板**：响应式 Web 界面，直观展示实时状态与审计日志。
